Form Builder Documentation

Getting Started

The Form Builder provides a declarative way to create dynamic, validated forms with support for multiple field types, async data loading, and real-time validation.

Field Types

Supported field types include:

  • text - Standard text input
  • email - Email input with validation
  • number - Numeric input with min/max
  • select - Dropdown with options (static or async)
  • textarea - Multi-line text area
  • switch - Toggle switch for boolean values
  • custom - Extensible custom field type

Basic Usage

const builder = new FormBuilder();
const form = builder.generateForm('myForm', {
  fields: [
    { name: 'email', type: 'email', label: 'Email', required: true }
  ],
  validation: {
    email: ['required', 'email']
  }
});
document.getElementById('container').innerHTML = form.render();

Validation Rules

Built-in validation rules:

  • required - Field must have a value
  • email - Must be valid email format
  • min/max - Numeric boundaries
  • minLength/maxLength - String length constraints
  • pattern - Regular expression match
  • custom function - Custom validation logic

Async Data Loading

formBuilder.loadAsyncOptions(formId, 'category', async () => {
  const res = await fetch('/api/categories');
  return res.json();
});

Form Submission

const form = builder.generateForm('myForm', {
  fields: [...],
  validation: {...}
});
const result = form.submit();
if (result.success) {
  console.log('Form values:', result.values);
}
